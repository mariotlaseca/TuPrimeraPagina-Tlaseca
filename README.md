# Proyecto TuPrimeraPagina+Tlaseca

Este proyecto sirve como una base para generar un sitio web de venta de joyeria en mayoreo en linea
Para utilizar este programa por favor revisar requierments.txt

La pagina cuenta con varias pesta√±as que permiten agregar diferentes tipos de productos, y nuevos productos a los tipos ya existentes.

Para utilizar la pagina, primero se debe de entrar en el html principal que no tiene ninguna designacion (""), de ahi podremos visualizar varios botones, de los cuales los unicos que son funcionales hasta el momento son home, catalogo y la barra de busqueda.

Daremos click en catalogo que es el boton que nos lleva al resto de opciones funcionales actuales, aqui encontraremos 7 botones, 1 para crear nuevos productos y otros 6 para crear articulos dentro de los productos ya existentes.
Debajo de estos botones podremos encontrar 6 botones mas grandes, en estos podremos acceder a visualizar que articulos hay en cada producto y revisar su informacion dispoible, dentro de este menu no se pueden realizar mas acciones.

Si hacemos click en crear algun nuevo articulo nos abrira una ventana con un form que debemos llenar para poder crear dicho articulo una ves creado, este nuevo articulo se vera reflejado en la ventana del producto al que pertenezca.
